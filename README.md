# Curso Lógica de Programação Java para Iniciantes

    Aqui é apresentado breve descritivo de cada código elaborado para resolução dos exercícios do curso de Java
    Em todos os exercícios foram incrementadas mais funcionalidades além das propostas nos enuciados, para desafiar o participante, e a certificação de que os conceitos apresentados foram realmente assimilados.

* Exercício 1 proposto para fixar os operadores matemáticos, tipos de variáveis utilizadas para armazenar números de variados formatos, como inteiros e decimais. Também apresentado os limites de cada tipo de variável do tipo numérica.

* Exercício 2 para leitura de dados utilizando Scanner. Podendo ser utilizado para leitura de dados do tipo string e numéricos, os quais foram armazenados em variáveis específicas e declaradas conforme o tipo a ser utilizado pelo código. Também foi utilizado o recurso de concatenação de strings, de modo a mostrar dados mais completos ao usuário.

* Exercício 3 proposto para validar aprovação de alunos, analisando as notas digitadas pelo usuário. Iniciado uso de estruturas de decisão, utilizando diferentes tipos de formato de dados, como, strings, números decimais (double), variáveis do tipo boolean e strings. Com isso, permitindo a manipulaçao (entrada, análise e apresentação de dados ao usuário) de variados tipos de dados. Verificado e estudado os tipos primitivos de variáveis, bem como declará-las.

* Exercício 4 para calcular valor de desconto de acordo com o valor inserido pelo usuário. Neste caso, utilizado utilizado comando if e manipulação de diversas variáveis, as quais foram envolvidas em cálculos e fórmulas. Estudado como escolher bons nomes de variáveis, não iniciar com números, não conter acentos e, a recomendação para declarar constantes.

* Exercício 5 proposto elaborar código para construir uma calculadora, no qual foi implementado o operador potenciação e usando os símbolos das operações matemáticas. Apresentado os operadores fundamentais e suas combinações, de modo a simplificar linhas de código. Operadores lógicos também foram estudados. Implementado uso de estruturas de decisão encadeada com estruturas de iteração, utilizando for para auxiliar na operação de potenciação, podendo elevar qualquer número a qualquer potência, como uma calculadora científica.

* Exercício 6 proposto para calcular gasto total mensal utilizando os operadores combinados (+=; -=; *=; /=). Mantendo o mesmo conceito de entrada de dados pelo usuário, uso de concatenação de dados e mostrando os resultados ao usuário. Testado os operadores combinados com uso de variáveis e números diretos aplicados às fórmulas.

* Exercício 7 elaborado código para validação de requisitos para se aposentar. Neste, implementado linhas de código para tratativas de varidas possibilidades, mostrando ao usuário se preenche os requisitos e o que falta para atender os requisitos e se aposentar. De um modo geral, cercar e trazer ao usuário dados completos para atender expectativas de utilizar apps.

* Exercício 8 proposto código para analisar e validar aprovação de candidatos em um concurso. Utilizando estruturas de tomada de decisão encadeadas e, também analisando possibilidades para report mais completo ao usuário.

* Exercício 9 exercício para calcular bônus de funcionário. Este código compara o realizado com a meta e dependendo da média salarial do funcionário, dertermina o bônus a ser pago ao funcionário. Exercício aplicando conceitos de if encadeado. Implementado mais uma possibilidade  para retorno ao usuário sobre o bônus do funcionário, caso o mesmo não atinja o limite inferior para 80% da meta.

* Exercício 10 proposto para aplicação dos conceitos da estrutura de decisão SWITCH, no qual foi solicitado pelo usuário inserir um número de 1 a 7, tendo a tratativa caso este esteja fora desse range, e como resultado, indica a qual dia da semana refere-se o número digitado. Primeiros comandos para tratativa de erros, caso dados sejam inseridos incorretamente, impedindo bug do código.
